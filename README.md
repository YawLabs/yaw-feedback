# Yaw — Feedback

Public issue tracker for [Yaw Terminal](https://yaw.sh) and MCP Hosting.

Use this repo to report bugs, request features, or share feedback.

## Products

- **Yaw Terminal** — modern terminal for Windows, macOS, and Linux. Website: [yaw.sh](https://yaw.sh)
- **MCP Hosting** — hosted MCP servers.

## File an issue

Pick the matching template from the [New issue](https://github.com/YawLabs/yaw-feedback/issues/new/choose) page:

- Bug — Yaw Terminal
- Feature — Yaw Terminal
- Bug — MCP Hosting
- Feature — MCP Hosting

## Before filing

- Search existing issues first — your report may already be tracked.
- Include your app version and OS for bugs.
- Screenshots and logs help a lot.
